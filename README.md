## - Installation & Requirements:
```bash
> go install github.com/erickfernandox/slicepathurl@latest
```
OR
```bash
> git clone https://github.com/erickfernandox/slicepathurl.git
> cd slicepathurl
> go build slicepathurl.go
> chmod +x slicepathurl
> ./slicepathurl -h
```
